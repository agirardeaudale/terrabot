============
Installation
============

Pip
---

.. code-block:: console

    $ pip install terrabot

From source
-----------

From the `Github repo`_:

.. code-block:: console

    $ git clone git://github.com/agirardeaudale/terrabot
    $ python setup.py install


.. _Github repo: https://github.com/agirardeaudale/terrabot
